Built for the Code for Philly 2016 Hackathon, this site seeks to empower votes to make informed decisions.

Pulled from the City of Philadelphia 2015 Open Data for campaign contributions, this site presents the information in a visual representation. For each position that was up for election, we have included every candidate that entered the race, resulting in politicians that lost in the primaries listed side by side with primary winners.

This project is an open source project, located on Git0Hub at PhillyDemHack. Communication for this project can be found on our Slack page.
